CSS units: CSS supports a wide range of units for specifying lengths, sizes, and other measurements.

1.Absolute Units: Represent fixed values. They do not change based on parent element, screen size or font-size.

>> Pixels (px): 1px is considerd as 1/96 of an inch
>> Centimeters (cm), millimeters(mm), inch(in): Basis of real world measurements.
>> Points (pt), picas(pc): Legacy approach


When to use?
For print layout

2.Relative Units: are based on relation with another element's size, viewport size.
Responsive design.
a) Font-related units:
>> em: relative to the font-size of the current element's parent
>> rem: root em: relative to the root element's(html) font size, not the parent
>> ex: Relative to the x-height


b) Percentage based units:
>> %(percent): relative to element's parent


c) viewport-relative units:
viewport: It is the visible area of web page in the browser window or on a device screen. The rectangular area/region where browser renders and shows content

>>vw: viewport width
1vw = 1% of viewport width

>>vh: viewport height
1 vh = 1% of viewport height

>>vmin:
1 vmin = 1% of the smaller dimension (width or height) of the viewport

>>vmax:
1 vmax= 1% of the larger dimensiom(width or height) of the viewport


When to use which unit?
>> px: Fine or precise values
>> rem: Font sizes or whenever spacing is needed, Responsive.
>> %: Great width or heights
>> vw, vh, vmin, vmax: Ideal for full-screen sections, is adapts to screen size.
>> cm,mm,in,pt: For print layout
