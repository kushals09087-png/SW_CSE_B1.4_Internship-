Position property: dictates how an element is placed on a webpage. It controls whether an element stays in the normal page flow, or moves relative to its neighbours or locks it in a place when user scrolls
It in a place when user scrolls
1. Static: Normal page flow. (Default)
2. Relative: Moves away from its normal spot without disturbing neighbours.
3. Absolute: Flies out of normalflow, positining itself inside its closest relative parent.
4. Fixed: Locks to the viewport. Stays in the same spot on the screen when scrolling.
5. Sticky: Scrolls naturally with the page, then locks in place like FIXED when hits the edge.

Both Display property & Position Property, controls layout, diffrence is that Display creates the box and Position move the Box.
