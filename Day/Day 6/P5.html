Display property: tells how an element behaves on a webpage. It controls how much spaces element takes up and where it sits in relation to surrounding elements.

4 types:
1.Block: Takes full width of the screen
2.Inline: It stays in the same line, ignores width or height.
3.Inline-block: Stays on the same line , accepts the width or height property
